# Rashail's Surprise 💖

A beautiful romantic surprise website built with React + Vite + Framer Motion.

## Setup

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

The `dist/` folder is ready to deploy to Vercel, Netlify, or any static host.

### Deploy to Vercel

1. Push this project to a GitHub repository
2. Import the repo on [vercel.com](https://vercel.com)
3. Vercel auto-detects Vite — click **Deploy**

### Deploy to Netlify

1. Run `npm run build`
2. Drag the `dist/` folder to [netlify.com/drop](https://app.netlify.com/drop)

## Photos

Replace the files in `public/` to update the memory gallery:
- `public/photo1.png` → `public/photo5.png`

## Music

The background music button links to "Khaab by Akhil" on YouTube via an embedded iframe. The music is muted by default — click 🎵 to unmute.
